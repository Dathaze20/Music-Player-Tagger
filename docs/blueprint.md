# **App Name**: Muzio AI

## Core Features:

- Muzio Player UI: Replicates the Muzio Player interface for music playback.
- Automatic Metadata Tagging: Analyzes audio files to automatically identify song metadata (title, artist, album, track number).
- Album Art Finder: Uses an AI tool to search for and apply album art to music files.
- Lyrics Integration: Scrapes lyrics from the web using AI for precise lyric matching, and displays them in sync with the music.
- Information Display: Displays album art, metadata, and synced lyrics within the Muzio Player interface.

## Style Guidelines:

- Use Muzio Player's original color scheme as the primary design.
- Accent color: Teal (#008080) to highlight selected elements and interactive components.
- Maintain the original font style of Muzio Player for consistency.
- Employ a flat design icon set that complements the Muzio Player aesthetic.
- Replicate the layout and spacing of Muzio Player for a familiar user experience.
- Use subtle animations for transitions and user interactions to enhance the user experience.
