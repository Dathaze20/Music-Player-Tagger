# Muzio AI - Smart Music Player

A pure HTML/CSS/JavaScript music player with AI-powered auto-tagging.

## How to Use

1. Download the 3 files: `index.html`, `style.css`, `app.js`
2. Open `index.html` in any browser or Web Code on Android
3. Tap the **+** button to import music files
4. Set your Gemini API key in the side menu > AI Settings

## Features

- Dark theme music player UI
- Artists, Songs, Albums, Playlists tabs
- AI auto-tagging via Google Gemini (artist, album, year, genre, release type)
- Album/Mixtape/EP/Single classification
- Batch album editing
- Search across all music
- No build tools needed - works directly in any browser
